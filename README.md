# Atividade 1 CSI-26/2023

SImples projeto contendo conjunto de páginas web com informações sobre a escola Centro Educacional 02 de Sobradinho/DF.

Foi empregado HTML e CSS puro (sem o auxilio de framework)

Principais dificuldades: Faltou um melhor domínio de ferramentas de designer para que houvesse um planejamento melhor da interface. Contudo, a simplicidade do que foi pedido permitiu que o processo iterativo fosse o suficiente para se chegar na entrega proposta.
